ZF2 Boilerplate App
==========

## Requires

* PHP >= 5.3
* [PHP Composer](https://getcomposer.org/)
* PostgreSQL or Mysql

## Installation

    #git clone https://github.com/paunin/zf2boilerplate.git yourProject && cd yourProject
    #./install.sh

@see `install.sh` to understand installation process
